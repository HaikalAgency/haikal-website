# Haikal Agency — Website Design Specification

## Project Goal

Design a premium, handcrafted website for **Haikal Agency** that feels like a Silicon Valley software company (Stripe, Linear, Vercel, Framer, Apple), not a freelance portfolio.

### Brand Personality

- Premium
- Minimal
- Engineering-driven
- Trustworthy
- Corporate
- Innovative
- Timeless

## Color Palette

- Background: #081A2F
- Secondary: #0E2342
- Primary: #155EEF
- Accent: #06D6F0
- White: #F8FAFC
- Gray: #94A3B8

## Typography

Headings: Space Grotesk
Body: Inter

Use large typography, generous whitespace, 8px spacing system, rounded corners (16–24px), subtle shadows, thin borders.

# AI Constraints

The website MUST NOT look AI-generated.

Avoid:

- Generic SaaS templates
- Stock illustrations
- Repeated card layouts
- Oversized glowing effects
- Excessive gradients
- Generic icons

Instead:

- Use asymmetric layouts.
- Every section should have a unique composition.
- Use custom geometric icons.
- Use visual storytelling.
- Plenty of whitespace.
- Architectural inspiration.
- Premium micro-interactions.
- Consistent spacing.
- Professional hierarchy.

Final impression:
"This is a premium international software engineering company capable of delivering enterprise-grade digital products."
